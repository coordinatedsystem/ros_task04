API参考
这里列出了所有模块:
[详情级别 12]
 版本信息接口获取SDK版本信息接口
 初始化及释放接口包含SDK资源初始化和释放接口
 采集卡接口包含采集卡的枚举、信息获取、打开和关闭等接口
 相机接口包含相机的枚举、信息获取、打开和关闭等接口
 图像采集接口包含流的信息获取、打开和关闭，注册和取消缓存等接口
 图像处理接口包含显示一帧图像、保存图像、像素格式转换等接口
 ▼通用接口包含XML接口、属性配置、读写相机文件以及读写寄存器接口
 XML接口包含获取XML文件接口
 属性配置接口包含属性配置相关接口
 读写相机文件接口包含文件读取和文件写入接口
 读写寄存器接口包含读取和写入寄存器接口
 事件接口包含注册事件回调函数和注册异常信息回调函数接口
 回调函数定义包含帧缓存信息回调函数、异常信息回调函数及事件信息回调函数
 结构体定义包含相关结构体定义
 枚举类型定义包含不同类型的枚举信息
 宏定义包含一些常量定义
 ▼状态码介绍SDK使用过程中不同状态码对应的值和说明
 正确码
 通用错误码范围为0x80190001-0x801900FF
 GenICam错误码范围为0x80190100-0x801901FF
 图像处理错误码范围为0x80190200-0x801902FF
 GenTL错误码范围为（-1001）-（-10000）
