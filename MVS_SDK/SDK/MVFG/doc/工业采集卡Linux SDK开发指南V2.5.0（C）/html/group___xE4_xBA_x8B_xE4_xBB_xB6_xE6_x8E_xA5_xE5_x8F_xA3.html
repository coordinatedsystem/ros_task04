◆ MV_FG_RegisterEventCallBack()

MV_FGCTRL_API int __stdcall MV_FG_RegisterEventCallBack ( IN PORTHANDLE  hPort,
IN const char *  strEventName,
IN MV_FG_EventCallBack  cbEvent,
IN void *  pUser 
)

注册事件回调函数

参数
hPort[IN] 参数句柄（可以使用采集卡/相机句柄）
strEventName[IN] 事件名称
cbEvent[IN] 事件回调函数
pUser[IN] 用户自定义数据
返回
成功，返回MV_FG_SUCCESS；失败，返回状态码
备注
在回调函数中处理耗时操作会阻塞后续事件信息的获取。
该接口适用用采集卡和相机事件回调，通过传入不同句柄进行注册。

◆ MV_FG_RegisterExceptionCallBack()

MV_FGCTRL_API int __stdcall MV_FG_RegisterExceptionCallBack ( IN PORTHANDLE  hPort,
IN MV_FG_ExceptionCallBack  cbException,
IN void *  pUser 
)

注册异常信息回调函数

参数
hPort[IN] 参数句柄（可以使用采集卡/相机句柄）
cbException[IN] 异常信息回调函数
pUser[IN] 用户自定义数据
返回
成功，返回MV_FG_SUCCESS；失败，返回状态码
备注
在回调函数中处理耗时操作会阻塞后续异常信息的获取。
该接口适用采集卡、相机异常回调，通过传入不同句柄进行注册。
目前不支持采集卡的掉线事件，采集卡不支持热插拔，不会出现掉线的情况；采集卡升级后必须要重启PC才能使用，已给出友好提示。
