控制采集卡

控制采集卡包括枚举采集卡、打开/关闭采集卡、配置采集卡。

枚举采集卡

通过三步完成采集卡的枚举：
  1. 调用 MV_FG_UpdateInterfaceList() 接口，输入nTLayerType参数枚举指定类型的采集卡，支持多种类型采集卡同时枚举，其返回参数pbChanged标识SDK内部的采集卡列表是否有更新，true表示列表有更新，false表示列表和之前一样。
  2. 调用 MV_FG_GetNumInterfaces() 接口，获取采集卡数量，其返回参数pnNumIfaces标识当前电脑上连接的指定类型采集卡的数量。假设该数量为N，则表示当前共有枚举到N个指定类型的采集卡，可以访问索引为[0，N-1]范围内的采集卡。
  3. 调用 MV_FG_GetInterfaceInfo() 接口，获取对应索引下的采集卡基本信息，并存储在 MV_FG_INTERFACE_INFO 结构体中返回给用户，用户可以通过该结构体中的nTLayerType字段区分不同类型的采集卡信息。
    下面的代码片段演示了枚举采集卡的过程：
    /*初始化运行环境*/
    MV_FG_Initialize();
    /*更新采集卡列表*/
    bool bChanged = false;
    nRet = MV_FG_UpdateInterfaceList(MV_FG_CXP_INTERFACE | MV_FG_GEV_INTERFACE | MV_FG_CAMERALINK_INTERFACE | MV_FG_XoF_INTERFACE, &bChanged);
    Check(nRet);
    /*获取采集卡数量*/
    unsigned int nInterfaceNum = 0;
    nRet = MV_FG_GetNumInterfaces(&nInterfaceNum);
    Check(nRet);
    for (unsigned int i = 0; i < nInterfaceNum; i++)
    {
    /*根据索引获取采集卡信息*/
    MV_FG_INTERFACE_INFO stInterfaceInfo = { 0 };
    nRet = MV_FG_GetInterfaceInfo(i, &stInterfaceInfo);
    Check(nRet);
    // TODO: 处理采集卡信息
    }
关闭所有指定类型的采集卡后，可以调用 MV_FG_ReleaseTLayerResource() 接口释放电脑上指定类型采集卡所占用的资源，一般是在用户不需要使用该类型采集卡或者在程序退出前调用：
/*释放指定类型的采集卡资源*/
nRet = MV_FG_ReleaseTLayerResource(MV_FG_CXP_INTERFACE | MV_FG_GEV_INTERFACE | MV_FG_CAMERALINK_INTERFACE | MV_FG_XoF_INTERFACE);
Check(nRet);

打开采集卡

  • 通过调用 MV_FG_OpenInterface() 接口，输入索引nIndex打开采集卡。打开成功后返回采集卡句柄，该句柄可用于采集卡的参数配置、相机枚举和事件获取等操作。
    /*打开采集卡*/
    IFHANDLE hInterface = NULL;
    nRet = MV_FG_OpenInterface(nInterfaceIndex, &hInterface);
    Check(nRet);
    // TODO：进行采集卡配置，相机枚举等操作
  • 通过调用 MV_FG_OpenInterfaceEx() 接口，输入索引nIndex，并指定打开权限后打开采集卡。打开权限包括只读权限和控制权限：只读权限，仅允许从采集卡中读取信息而不允许进行写入；控制权限，既允许从采集卡中读取信息，也允许进行写入。
    /*打开采集卡，可指定权限*/
    IFHANDLE hInterface = NULL;
    nRet = MV_FG_OpenInterfaceEx(nInterfaceIndex, MV_FG_ACCESS_CONTROL, &hInterface);
    Check(nRet);
    // TODO：进行采集卡配置，相机枚举等操作
  • 通过调用 MV_FG_OpenInterfaceByID() 接口，输入采集卡ID，并指定打开权限后打开采集卡。使用该接口打开采集卡可以跳过采集卡枚举流程，但需能提前获取采集卡ID。采集卡ID是由型号、序列号和PCIe序号组成，中间通过“-”分隔，该ID可以是 MV_FG_INTERFACE_INFO 结构体中的chInterfaceID。同时输入采集卡ID时也支持模糊搜索功能，只需输入采集卡ID中能唯一标识该采集卡的信息即可，比如序列号或者PCIe序号。
    /*通过采集卡ID打开采集卡，可指定权限*/
    IFHANDLE hInterface = NULL;
    nRet = MV_FG_OpenInterfaceByID("SerialNumber", MV_FG_ACCESS_CONTROL, &hInterface);
    Check(nRet);
    // TODO：进行采集卡配置，相机枚举等操作
  • 通过调用 MV_FG_OpenInterfaceWithConfig() 接口，输入索引nIndex，并导入配置文件打开采集卡，目前配置文件来自 MV_FG_FeatureSave() 接口。导入配置文件操作非必须，支持pcConfigFile传NULL，可自行选择是否导入。
    /*打开采集卡，可导入配置文件*/
    IFHANDLE hInterface = NULL;
    nRet = MV_FG_OpenInterfaceWithConfig (nInterfaceIndex, "ConfigFileName", &hInterface);
    Check(nRet);

关闭采集卡

当需要关闭采集卡时，可以调用 MV_FG_CloseInterface() 接口，输入对应采集卡句柄进行关闭。应当注意的是，当采集卡被关闭后，原先指向采集卡的句柄将失效，不能再用于其它接口的调用。
/*关闭采集卡*/
nRet = MV_FG_CloseInterface(hInterface);
hInterface = NULL;
Check(nRet);
