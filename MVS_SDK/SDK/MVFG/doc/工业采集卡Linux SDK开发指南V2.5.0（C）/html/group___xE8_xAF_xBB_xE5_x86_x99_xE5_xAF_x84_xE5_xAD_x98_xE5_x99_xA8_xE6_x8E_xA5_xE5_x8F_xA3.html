◆ MV_FG_ReadPort()

MV_FGCTRL_API int __stdcall MV_FG_ReadPort ( IN PORTHANDLE  hPort,
void *  pBuffer,
int64_t  nAddress,
int64_t  nLength,
int64_t *  pnOutLength 
)

读取采集卡/相机寄存器

参数
hPort[IN] 句柄（可以使用采集卡/相机句柄）
pBuffer[IN] 作为返回值使用，保存读到的内存值（注意根据采集卡/相机类型区分大小端）
nAddress[IN] 读取的内存地址，该地址可以从采集卡/相机的XML文件中获取，形如xxx_RegAddr的节点值
nLength[IN] 读取的内存长度
pnOutLength[OUT] 实际返回的内存长度
返回
成功，返回MV_FG_SUCCESS；失败，返回状态码
备注
访问采集卡/相机，读取某段寄存器的数据。

写入采集卡/相机寄存器

参数
hPort[IN] 句柄（可以使用采集卡/相机句柄）
pBuffer[IN] 待写入的内存值（注意根据采集卡/相机类型区分大小端）
nAddress[IN] 待写入的内存地址，该地址可以从采集卡/相机的XML文件中获取，形如xxx_RegAddr的节点值
nLength[IN] 待写入的内存长度
pnOutLength[OUT] 实际写入的内存长度
返回
成功，返回MV_FG_SUCCESS；失败，返回状态码
备注
访问采集卡/相机，把一段数据写入某段寄存器。
