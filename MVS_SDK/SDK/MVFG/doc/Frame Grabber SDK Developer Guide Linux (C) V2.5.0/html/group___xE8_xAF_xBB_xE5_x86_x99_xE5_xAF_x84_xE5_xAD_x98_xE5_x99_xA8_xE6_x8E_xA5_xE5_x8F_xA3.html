MV_FGCTRL_API int __stdcall MV_FG_ReadPort ( IN PORTHANDLE  hPort,
void *  pBuffer,
int64_t  nAddress,
int64_t  nLength,
int64_t *  pnOutLength 
)

Read frame grabber or camera register.

Parameters
hPort[IN] Handle (frame grabber or camera).
pBuffer[IN] Used as a returned value to save the read-in memory value. (Memory value is stored in accordance with the big end and little end model.)
nAddress[IN] Memory address to be read, which can be obtained from the XML file of the frame grabber or camera. It is a node value in the format of xxx_RegAddr.
nLength[IN] Length of the memory to be read.
pnOutLength[OUT] Actual length of the returned memory.
Returns
Return MV_FG_SUCCESS for success, and return corresponding Error Code for failure.
Remarks
Access the frame grabber or camera and read data from a certain segment of the register.
