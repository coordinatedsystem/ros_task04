Sample Code
 
The console sample codes are listed below.
File Name Operation Description
GrabImage_CallbackAcquire Image via CallbackThe sample code shows how to use the SDK to set the internal buffer and call MV_FG_RegisterFrameCallBack() to acquire images. Time-consuming operations are not recommended in the callback function in case of frame loss.
GrabImageActively Acquire ImageThe sample code shows how to use the SDK to set the internal buffer and call MV_FG_GetFrameBuffer() to acquire images. How to call MV_FG_SaveBitmap() to save Bitmap images is demonstrated.
GrabImage_UserBufferAcquire Image via Cache RegistrationThe sample code shows how to register external buffer, use external buffer, and call MV_FG_GetImageBuffer() to acquire images. The external buffer reduces the times of copying data.
EventGet Frame Grabber EventThe sample code shows how to call MV_FG_RegisterEventCallBack() to get the event information (ReceiveImageFrameStart0) of images received by frame grabber.
ChunkDataGet Chunk DataThe sample code shows how to call MV_FG_GetBufferChunkData() to get chunk data and how to parse the exposure and timestamp of chunk data.
ConvertPixelTypeConvert Pixel FormatThe sample code shows how to call MV_FG_ConvertPixelType() to convert the picture format. In the sample code, images in Mono format are all converted to ones in Mono8 format, and colored images are all converted to ones in RGB8 format.
HBDecodeLossless DecodingThe sample code shows how to call MV_FG_HB_Decode() to decode image losslessly.
GrabMultipleCamera_CallbackAcquire Images from Multiple CamerasThe sample code shows how to acquire images from multiple cameras on a frame grabber.
ReconnectReconnect to CameraThe sample code shows how to restore acquiring images if the camera is disconnected.
FeatureSaveSave Device FeaturesThe sample code shows how to import/export parameters.
