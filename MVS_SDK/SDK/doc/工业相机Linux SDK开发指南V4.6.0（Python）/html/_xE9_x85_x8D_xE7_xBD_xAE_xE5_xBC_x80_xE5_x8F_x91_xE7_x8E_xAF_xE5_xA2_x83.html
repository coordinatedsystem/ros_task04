配置开发环境

在您开始二次开发前，请参照本文配置开发环境，将SDK引入至您的项目。

头文件和库文件

在安装软件客户端时，安装程序会将SDK所需的头文件和库文件安装到计算机上。SDK开发包路径: /opt/(软件名称)/Samples/(架构)/Python。
头文件路径：$(MVCAM_COMMON_RUNENV)\Samples\Python\MvImport
头文件文件名 描述
CameraParams_const.py 常量，类定义
CameraParams_header.py 结构体定义
MvCameraControl_class.py 接口文件
MvErrorDefine_const.py 状态码定义
PixelType_header.py 像素格式定义
MvISPErrorDefine_const.py ISP错误码定义
SDK提供Python程序库文件（“libMvCameraControl.so”），具体路径：${MVCAM_COMMON_RUNENV}/(架构)/libMvCameraControl.so。

开发环境要求

在使用工业相机SDK开发您的程序前，您需首先搭建Python编程环境。
注解
推荐使用Python3.X版本。您可在控制台中，输入“python –version”查看当前安装的Python版本，以确保Python环境安装正常。

引用SDK头文件

您可参照如下代码示例，在您的程序中引用SDK头文件。
#Python SDK 头文件路径
sys.path.append("../../MvImport")
from MvCameraControl_class import *
