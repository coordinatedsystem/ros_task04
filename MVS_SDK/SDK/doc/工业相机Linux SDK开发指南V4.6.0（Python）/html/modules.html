 相机相关配置包含枚举相机、创建句柄、打开相机等接口
 采集卡相关配置包含枚举采集卡、创建句柄、打开采集卡等接口

 读写相机文件包含从设备读取文件、将文件写入设备等接口
 设备升级包含设备升级和获取升级进度接口
 图像采集与渲染包含注册缓存、获取图像、渲染图像等接口

 事件及异常包括事件及异常接口
