错误处理和状态码

在您在程序中调用SDK接口后，接口会返回一个代表int型的状态码。状态码以16进制的形式呈现，其格式为“0x%x”。
如果接口调用正常，且未检测到任何错误，则会返回“0x00000000”，代表“成功，无错误”；否则，接口将返回错误码。
注解
您可在“MvErrorDefine_const.py”文件或 状态码 中，查找错误码及其对应定义和解释。
您可以参照如下示例，在您的程序中检查接口异常并返回错误信息。
# ch:打开设备 | en:Open device
ret = cam.MV_CC_OpenDevice(MV_ACCESS_Exclusive, 0)
if ret != 0:
print ("open device fail! ret[0x%x]" % ret)
sys.exit()
该例演示了打开相机时，获取 MV_CC_OpenDevice() 的调用情况。
  • 若能够正常打开相机，则该例不会返回报错信息。

  • 若调用接口时，相机设备已连接至第三方软件，则 MV_CC_OpenDevice() 无法正常打开相机。此时，该接口会返回如下信息。

    Open Device fail! nRet [0x80000203]

    在获得错误信息和对应状态码后，您可在状态码 或“MvErrorDefine_const.py”错误码文件中，查找对应内容。该例中，“0x80000203”代表“设备无访问权限”。
