示例程序概览

安装工业相机客户端后，您可从客户端的安装路径获取对应的示例程序文件。您可参照提供的示例，在您的程序中实现对应功能。
注解
此处仅介绍控制台示例程序。您可前往安装路径下的.\Development\Samples\Python以获取全部示例。
示例文件夹内包含如下类别的示例：
  • 基础示例
  • 采集卡示例
  • 面阵相机示例
  • 线阵相机示例

基础示例

以下为基础示例，您可从Samples文件夹下的General文件夹中获取对应的示例。
相机连接基础示例如下：
文件 示例 描述
ConnectSpecCamera.py 通过IP地址连接相机 演示通过相机IP地址和相机对应的网卡IP地址，实现将相机连接至您的程序。
注解
输入的IP地址格式为xx.xx.xx.xx。


图像获取基础示例
文件 示例 描述
GrabImage.py 获取相机图像 演示通过主动取图的方式获取相机图像。
Grab_Callback.py 通过回调获取相机图像 演示通过回调的方式获取相机图像。


图像处理基础示例
文件 示例 描述
ConvertPixelType.py 图像格式转换 演示如何通过图像格式转换接口将设备采集到的原始图像数据转换成所需的像素格式。
HighBandwidthDecode.py 高清无损压缩 演示通过高清无损压缩算法将从相机取到的无损压缩码流解码成裸数据。
ImageSave.py 保存相机设备图像 演示如何保存从相机设备拿到的图像。可保存的图像格式包含裸图(.raw)、jpeg、bmp、tiff、png。


参数设置基础示例
文件 示例 描述
MultiCast.py 设置组播模式 演示如何设置传输模式为组播模式。通过询问用户启动多播控制应用程序或者多播监控应用程序，并以指定的访问权限打开相机并设置组播模式。
ParametrizeCamera_FileAccess.py 导入或导出文件到相机 演示如何导入文件到相机、导出相机文件、以及获取文件存取进度。
ParametrizeCamera_LoadAndSave.py 导入或导出相机属性 演示如何将相机属性树的设置导出到.ini文件（例如，FeatureFile.ini）和从.ini文件导入到相机属性树中。


事件基础示例
文件 示例 描述
Events.py 获取相机事件 演示配置相机事件功能、注册事件回调以及在事件回调函数中处理获取到的事件信息。Event事件例如：ExposureEnd。


采集卡示例

以下为采集卡基础示例，您可从Samples文件夹下的FrameGrabber文件夹中获取对应的示例。
文件 示例 描述
Events_Interface.py 打开并接受采集卡事件 演示打开并接收采集卡事件。
InterfaceAndDevice.py 采集卡下的相机控制 演示通过普通枚举接口枚举到自研采集卡下的相机，并实现打开相机、开始取流等操作。
InterfaceDemo.py 采集卡的枚举及控制 演示通过枚举接口枚举自研采集卡，并实现打开采集卡、设置属性等操作。


面阵相机示例

以下为面阵相机示例，您可从Samples文件夹下的AreaScanCamera文件夹中获取对应的示例。
文件 示例 描述
ParametrizeCamera_AreaScanIOSettings.py 面阵相机IO设置 演示如何配置面阵相机触发相关节点IO。


线阵相机示例

以下为线阵相机示例，您可从Samples文件夹下的LineScanCamera文件夹中获取对应的示例。
文件 示例 描述
LineScanSoftwareTrigger.py 线阵相机软触发 演示设置线阵相机采用软触发功能。
MultiLightCtrl_ImageStitching.py 线阵相机分时曝光并拆图拼图 演示设置线阵相机采用2组分时曝光参数，先拆图再自上而下拼图，最后将拼好的raw图以bmp格式保存至本地。
ParametrizeCamera_LineScanIOSettings.py 线阵相机IO设置 演示配置线阵相机帧触发和行触发相关节点IO。
