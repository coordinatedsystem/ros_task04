获取相机图像

该示例程序说明如何通过主动取图的方式获取相机图像。创建句柄，打开相机，开始取流，创建取流线程（用户分配采集buffer， MV_CC_GetOneFrameTimeout() 取图（拷贝）），停止取流，关闭相机，销毁句柄。
