def MvCameraControl_class.MV_CC_RegisterExceptionCallBack (   self,
  ExceptionCallBackFun,
  pUser 
)

注册异常消息回调，在打开设备之后调用

参数
handle[IN] 设备句柄
cbException[IN] 异常回调函数指针
pUser[IN] 用户自定义变量
返回
成功，返回MV_OK；失败，返回状态码
备注
  • 该接口需要在 MV_CC_OpenDevice() 打开设备之后调用。
  • 设备异常断开连接后可以在回调里面获取到异常消息，GigE设备掉线之后需要先调用 MV_CC_CloseDevice() 接口关闭设备，再调用 MV_CC_OpenDevice() 接口重新打开设备。
