◆ SetBayerCvtQuality()

Int32 IPixelTypeConverter.SetBayerCvtQuality ( CFAMethod  method)

Sets image interpolation algorithm type.

Parameters
methodIt refers to the image interpolation method.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
It can be called to set the Bayer interpolation smoothing enabling parameter for the internal image conversion API. The interpolation algorithm used by IPixelTypeConverter and IImageSaver are specified by this API.

◆ SetBayerGammaValue()

Int32 IPixelTypeConverter.SetBayerGammaValue ( float  gammaValue)

Sets the Gamma value in Bayer pattern.

Parameters
gammaValueIt refers to the Gamma value, range: [0.1, 4.0].
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
The Gamma value set by this API will be used when calling IPixelTypeConverter and IImageSaver to convert Bayer 8/10/12/16 format to RGB 24/48, BGR 24/48, RGBA 32/64, or BGRA 32/64.

◆ SetGammaValue()

Int32 IPixelTypeConverter.SetGammaValue ( MvGvspPixelType  pixelType,
float  gammaValue 
)

Sets Gamma value in Mono 8 or Bayer 8/10/12/16 pattern.

Parameters
pixelTypeIt refers to the pixel format.
gammaValueIt refres to the Gamma value: [0.1, 4.0].
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
The Gamma value in Mono 8 pattern set via this API will be used when ConvertPixelType(MvCameraControl.IImage, byte[], out ulong, MvCameraControl.MvGvspPixelType) is called to convert Mono 8 to Mono 8.
The Gamma value in Bayer8/10/12/16 pattern set via this API will be used when IPixelTypeConverter and IImageSaverare called to convert Bayer 8/10/12/16 to RGB 24/48, RGBA 32/64, BGR 24/48, or BGRA 32/64.
This API is compatible with SetBayerGammaValue, and it supports Mono 8 pixel format.

◆ SetBayerCCMParam()

Int32 IPixelTypeConverter.SetBayerCCMParam ( CCMParam  ccmParam)

Enables and disables CCM and sets CCM parameters of Bayer pattern. The default quantitative scale is 1024.

Parameters
ccmParamIt refres to the CCM parameter.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
After the API is called to enable CCM and set the CCM, the CCM parameters will take effect when IPixelTypeConverter or IImageSaver is called to convert Bayer 8/10/12/16 format to RGB 24/48, RGBA 32/64, BGR 24/48, or BGRA 32/64.
