API Reference
Here is a list of all modules:
 SDK InitializationIncludes APIs for initializing and releasing SDK resources
 Camera ControlIncludes APIs and data structures for camera control
 Frame Grabber ControlIncludes APIs and data structures for frame grabber control
 Parameter ConfigurationIncludes APIs and data structures for parameter configuration of cameras and frame grabbers
 Image AcquisitionIncludes APIs and data structures for image acquisition
 Image RenderingIncludes APIs and data structures for image rendering (displaying acquired images on the application)
 Image ProcessingIncludes APIs and data structures for image processing (image decoding, image rotating, contrast adjusting, etc.)
 Event FunctionIncludes APIs and data structures for device event subscription and receiving
 Serial Port ControlIncludes APIs for serial port control
 GenTLIncludes APIs and data structures related to GenTL standard (enumerating third-party devices via GenTL APIs)
 Status CodesIncludes status code of API calling and exception messages returned from SDK
