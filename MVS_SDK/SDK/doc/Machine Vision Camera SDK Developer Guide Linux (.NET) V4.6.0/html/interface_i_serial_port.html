◆ Open()

int ISerialPort.Open ( )

Opens the serial port.

Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.

This API is supported by serial port devices.

◆ Read()

int ISerialPort.Read ( byte []  buffer,
int  length,
out uint  readLen,
uint  timeout 
)

Reads data from the serial port.

Parameters
bufferIt refers to the data buffer.
lengthIt refers to the length of data to be read.
readLenIt refers to the length of read data.
timeoutIt refers to the timeout duration, unit: ms.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
The API is in blocking mode, returning the result only when data is received, times out, or exception occurs.

Writes data to the serial port, allowing a maximum of 1024 bytes written at a time.

Parameters
bufferIt refers to the data buffer.
lengthIt refers to the length of data to be written.
writeLenIt refers to the length of written data.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
The API is in blocking mode, returning the result only when all data has been successfully transmitted or transmission fails.
