Int32 IImageDecoder.HBDecode ( IFrameOut  inFrameInfo,
out IFrameOut  outFrameInfo 
)

Decodes lossless compression stream into raw data.

Parameters
inFrameInfoIt refers to the input image and frame information.
outFrameInfoIt refers to the output image and frame information. After using the image, call Dispose to release image buffer to prevent memory constraints.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
It supports decoding the lossless stream of camera to RAW data and parsing the watermark of real-time images for the current camera.
Attention
  • If the input lossless stream is not real-time or does not belong to the current camera, an exception may occur during watermark parsing.
  • If the decoding failed, check if it is one of the following circumstances:
    1. Check if CPU supports SSE AVX command set.
    2. Check if there is an exception on the current frame (e.g., request timeout). If there is an exception, a decoding failure may occur.
    3. Check if there is an exception on image generating from camera. Exception occurs on image generating from camera, even without request timeout.
Examples
Image_HBDecode.cs, Image_Save.cs, MultiLightCtrl.cs, and MultiLightCtrl_ImageStitching.cs.
