Enumerates interfaces via GenTL and enumerates the camera connected to the interface via interface information. Here "interfaces" refer to frame grabbers or interfaces defined under GenTL standard.

Parameters
IFInfoIt refers to the interface information.
devListIt refers to the camera list.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
After device is enumerated, camera instance can be created via DeviceFactory.

◆ UnloadGenTLLibrary()

static Int32 GenTLManager.UnloadGenTLLibrary ( String  ctiPath)
static

Unloads CTI library.

Parameters
ctiPathIt refers to the path of the CTI file that is loaded during device enumeration.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Attention
Make sure that all devices enumerated by the CTI file are turned off before calling this API. Otherwise, MV_E_PRECONDITION will be returned.
