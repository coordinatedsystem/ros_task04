SDK Initialization

Before introducing the .NET assembly namespace to your project, initialize the SDK resources first.

Prerequisite

Configured the development environment as required, and introduced the SDK .NET assembly and its namespace. See details in SDK .NET Assembly Introduction.

Initialize SDK Resources

Call SDKSystem.Initialize() to initialize the SDK running environment, so as to apply for and distribute needed resources in advance, and improve the subsequent API calling fluency.
When you exit the application, you can call SDKSystem.Finalize() to finalize the SDK and release the system resources occupied by the SDK.
The following sample code shows the process of initializing the running environment.
// Initialize SDK resources
SDKSystem.Initialize()
// TODO: You can perform operations such as device control and image acquisition.
// Release SDK resources
SDKSystem.Finalize()
