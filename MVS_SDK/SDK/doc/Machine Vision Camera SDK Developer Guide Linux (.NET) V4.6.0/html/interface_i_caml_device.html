Detailed Description

Includes specific APIs and parameters of camera link serial port cameras.

Member Function Documentation

◆ SetDeviceBaudrate()

Int32 ICamlDevice.SetDeviceBaudrate ( CameraLinkBaudrate  baudrate)

Sets baud rate for the devices.

Parameters
baudrateIt refers to the baud rate.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Attention
The max. baud rate for camera link devices should be less than 115200. High baud rate may cause communication exception due to factors such as hardware specification, system configuration, and external interference.
