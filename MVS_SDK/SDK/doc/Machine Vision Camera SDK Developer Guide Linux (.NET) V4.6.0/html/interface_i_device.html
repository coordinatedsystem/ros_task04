◆ Open() [2/2]

Int32 IDevice.Open ( DeviceAccessMode  AccessMode,
UInt32  switchoverKey 
)

Turns on the device via specified access mode, only supported by GigE devices. For access mode details, refer to DeviceAccessMode.

Parameters
AccessModeIt refers to device access modes of the current process. Device access permission of the current process varies in different access modes.
switchoverKeyIt refers to the secret key for changing device access permission.
Returns
Returns MV_OK for success, and returns Error Code for failure.
Attention
MV_ACCESS_ExclusiveWithSwitch, MV_ACCESS_ControlWithSwitch, MV_ACCESS_ControlSwitchEnable, and MV_ACCESS_ControlSwitchEnableWithKey modes are not supported.
