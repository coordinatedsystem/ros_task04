◆ DisplayOneFrame() [1/3]

Int32 IImageRender.DisplayOneFrame ( IntPtr  hWnd,
IImage  image,
RenderMode  mode = RenderMode.Default 
)

Displays one frame of image (API 1).

Parameters
hWndIt refers to the windows handle.
imageIt refers to the image information.
modeIt refers to the rendering mode.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
For rendering mode set to RenderMode.OPENGL, rendering of images over 4 GB (in PixelType_Gvsp_RGB8_Packed, PixelType_Gvsp_BGR8_Packed, and PixelType_Gvsp_Mono8 format only) is supported.
For rendering mode set to RenderMode.D3D, the max. supported resolution is 16384 × 163840.
Examples
MultiLightCtrl.cs.
