◆ InvalidateNodes()

Int32 IParameters.InvalidateNodes ( )

Clears the buffer of the GenICam node.

Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
GenICam configuration file is read during loading machine vision camera node, and this API can be called to clear the GenICam buffer.

◆ ReadMemory()

Int32 IParameters.ReadMemory ( Int64  address,
Int64  length,
out Byte []  buffer 
)

Reads data from register by device register address.

Parameters
addressIt refers to the memory address to be written to. It can be acquired from Camera.xml file, in a form similar to XML node value of xxx_RegAddr.
lengthIt refers to the length of memory to be read.
bufferIt refers to the data buffer for saving memory value that is read (GEV memory value is stored based on big-endian mode, and memory value of other devices is stored based on little-endian mode).
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
Access device and read the data from certain register.
