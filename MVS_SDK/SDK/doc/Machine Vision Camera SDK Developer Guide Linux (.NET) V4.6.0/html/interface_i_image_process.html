Detailed Description

Includes APIs of image processing, such as rotation, flipping, contrast, and resolution.

Member Function Documentation

◆ RotateImage() [1/2]

Int32 IImageProcess.RotateImage ( IImage  inImage,
out IImage  outImage,
ImageRotateAngle  angle 
)

Rotates images (API 1).

Parameters
inImageIt refers to the input image.
outImageIt refers to the output image. After using the image, call Dispose to release image buffer to prevent memory constraints.
angleIt refers to the rotation angle.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Attention
This API only supports 90°, 180°, and 270° rotation of images in Mono 8, RGB 24, and BGR 24 formats.
