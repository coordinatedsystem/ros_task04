Overview

Machine Vision Camera SDK (MvCameraSDK) is designed to implement the management of multiple different frame grabbers, machine vision cameras, light controllers, and board type light controllers. You can set up the vision applications based on the SDK that meet your needs to realize device control, parameter configuration, data acquisition, data saving, and other functions.
This SDK is compatible with GigE vision protocol, USB3 vision protocol, camera link protocol, CoaXPress protocol, and XoFLink protocol.

Development Environment

Linux Operating System (64-bit): Ubuntu 16.04 to 24.04 (64-bit), Redhat7 (64-bit), Centos7 (64-bit) or later.
Note
For frame grabbers, only 64-bit Ubuntu 16.04 to 24.04 are supported.
.Net Running Environment Requirements: .Net 5 or later version.

Development Notice

The Machine Vision Camera client is installed in /opt/software name by default. You can customize the path as needed.
The installation path contains the Development folder, which contains the dependent files of the SDK as follows:
  • bin (compiled deliverable)
  • doc (documents for secondary development, such as SDK and user manual)
  • includes (header files)
  • lib (DLL)
  • Samples (sample programs in different programming languages and .Net assembly)
Attention
Files under development directories of different client version might vary. The actual circumstance shall prevail.
Machine Vision Camera SDK Linux Version includes driver and logserver (log service). You can acquire them in the installation path.
