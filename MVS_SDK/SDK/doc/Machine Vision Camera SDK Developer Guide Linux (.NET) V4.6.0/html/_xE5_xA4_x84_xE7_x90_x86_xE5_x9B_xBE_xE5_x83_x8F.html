Image Processing

The SDK provides abundant image processing APIs for displaying image, saving image, converting pixel format, reforming image, recording, etc. After images are acquired (IFrameOut.Image), they can be transmistted into these APIs for image processing.

Display Image

IDevice provides the IImageRender feature, for displaying images.
After images are acquired, call IImageRender.DisplayOneFrame() to display the image in the specified window.
// Image information
IFrameOut frameOut;
// Get image information
nRet = device.StreamGrabber.GetImageBuffer(1000, out frameOut);
if (MvError.MV_OK == nRet)
{
// Display image
device.ImageRender.DisplayOneFrame(pictureBox1.Handle, frameOut.Image);
// Release image
device.StreamGrabber.FreeImageBuffer(frameOut);
}

Save Image

IDevice provides the IImageSaver feature, for saving images.
After images are acquired, you can call IImageSaver.SaveImageToFile() to save images as local files, and call IImageSaver.SaveImageToBuffer() to save files to the Byte array.
The following sample code shows how to saves images as local BMP files.
// Saving path of the image file
string imagePath = "Image.bmp";
// Image information
IFrameOut frameOut;
// Get image information
nRet = device.ImageSaver.GetImageBuffer(1000, out frameOut);
if (MvError.MV_OK == nRet)
{
// File format information for saving
ImageFormatInfo imageFormatInfo = new ImageFormatInfo();
imageFormatInfo.FormatType = ImageFormatType.Bmp;
// Save image
device.ImageSaver.SaveImageToFile(imagePath, frameOut.Image, imageFormatInfo, CFAMethod.Optimal);
// Release image
device.StreamGrabber.FreeImageBuffer(frameOut);
}

Convert Pixel Format

IDevice provides the IPixelTypeConverter feature, for converting image format.
You can call IPixelTypeConverter.ConvertPixelType() to convert images to other pixel formats.
For example, convert Bayer format to RGB/BGR format. The interpolation to images in Bayer format supports several algorithms such as smoothing filter, Gamma correction, and CCM correction. See details in IPixelTypeConverter.
The following sample code shows how to convert images to RGB Format.
IImage outImage;
// Target pixel format
MvGvspPixelType dstPixelType = MvGvspPixelType.PixelType_Gvsp_RGB8_Packed;
// Convert pixel format
result = device.PixelTypeConverter.ConvertPixelType(inputImage, out outImage, dstPixelType);
if (result != MvError.MV_OK)
{
Console.WriteLine("Image Convert failed:{0:x8}", result);
return;
}
Console.WriteLine("Image Convert success!");
// Release image buffer after using for avoiding frequent garbage collection (GC) due to rapid memory increase
outImage.Dispose();

Decode Image

IDevice provides the IImageDecoder feature, currently for decoding losslessly compressed images into raw data.
Call IImageDecoder.HBDecode() to decode the compressed image data sent by camera into complete raw data.
Attention
Check if the input image is complete before decoding. If some data packet of image is lost, the API will return the failure message.
IFrameOut frameOut;
// Get one frame of image
result = device.StreamGrabber.GetImageBuffer(1000, out frameOut);
if (MvError.MV_OK == result && frameOut.LostPacket == 0)
