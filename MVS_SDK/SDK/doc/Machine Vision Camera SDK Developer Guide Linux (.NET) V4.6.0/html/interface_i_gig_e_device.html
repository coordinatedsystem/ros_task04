◆ ForceIp()

Int32 IGigEDevice.ForceIp ( UInt32  ip,
UInt32  subnetMask,
UInt32  defaultGateway 
)

Forces the configuration of device network parameters, including IP address, subnet mask, and default gateway.

Parameters
ipIt refers to the IP address.
subnetMaskIt refers to the subnet mask.
defaultGatewayIt refers to the default gateway.
Returns
Returns MV_OK for success, and returns Error Code for failure.
Attention
  • After forcing the configuration of device network parameters, create device handle again. This API is only supported by GigE vision devices.
  • The device will restart after you call this API to set network parameters forcefully when the device is in DHCP status.
Examples
GigE_ForceIP.cs.

◆ SetResendMaxRetryTimes()

Int32 IGigEDevice.SetResendMaxRetryTimes ( UInt32  retryTimes)

Sets the max. command resending times.

Parameters
retryTimesIt refers to the max. command resending times (20 by default).
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
This API should be called after resending packet is enabled via calling SetResend. If APIs are not called in order, SetResendMaxRetryTimes() calling will be failed and MV_E_CALLORDER will be returned.
