Samples

Descriptions of all sample code files of the current version are as follows.
Note
Get the corresponding sample code file from .\Development\Samples\C#\MvCameraControlNet_V2 under the client installation path

Sample Programs of Camera Connection

File Name Sample Program Description
GigE_ConnectCameraByIP.cs Connect to GigE Camera by IP Address It shows how to connect GigE camera to your application based on GigE camera IP address.
Reconnect.cs Reconnect to Camera It shows how to reconnect to camera when connection errors occur.
InterfaceAndDevice.cs Enumerate Camera by Frame Grabber It shows how to enumerate cameras and grab images with the frame grabber after turning on it.

Sample Programs of Event Receiving

File Name Sample Program Description
Events_Camera.cs Receive Camera Event It shows how to receive camera events.
Events_Interface.cs Receive Frame Grabber Event It shows how to receive frame grabber events.

Sample Programs of Image Acquisition

File Name Sample Program Description
Grab_Callback.cs Get Image via Callback It shows how to get images via callback (FrameGrabedEvent). In this way, the image grabbing thread will start inside the SDK, and will call back images acquired by camera to your application via event.
ParametrizeCamera_AreaScanIOSettings.cs Configure IO Settings of Area Scan Camera It shows how to configure trigger node IO of area scan camera.
ParameterCamera_LineScanIOSettings.cs Configure IO Settings of Line Scan Camera It shows how to configure node IO settings of frame trigger and line trigger for line scan cameras and get images via callback. In this way, the image grabbing thread will start inside the SDK, and will call back images acquired by camera to your application via event.
Grab_GetImageBuffer.cs Get Image via Polling It shows how to create the image acquisition thread, and call IStreamGrabber.GetImageBuffer() / IStreamGrabber.FreeImageBuffer() via polling to get and release images acquired by camera.
Grab_Strategy.cs Configure Image Acquisition Strategy It shows how to achieve image acquisition strategy.
Grab_ActionCommand.cs Get Image via ActionCommand Trigger It shows how to trigger the GigE vision camera to acquire images via the ActionCommand.
Grab_GigEMultiCast.cs MultiCast It shows how to realize the MultiCast function of GigE vision camera on your application.
GigE_ForceIP.cs Set Static IP Address for GigE Vision Camera It shows how to modify the IP address of GigE vision camera on your application.
MultiLightCtrl.cs Multi-Light Control It shows how to get and rebuild images from camera that supports the multi-light control.
MultiLightCtrl_ImageStitching.cs Multi-Light Control of Line Scan Camera It shows how to get and rebuild images from line scan camera that supports the multi-light control.
Grab_ImageClone.cs Deep Copy It shows how to deep copy images during image acquisition and put images into queue for asynchronous processing.
Grab_ChunkData.cs Get Chunk Data It shows how to get images via callback (FrameGrabedEvent). In this way, the image grabbing thread will start inside the SDK, and will call back images acquired by camera to your application via event.
ParameterInterface_SoftwareTrigger.cs Frame Grabber Software Trigger It shows how to achieve software trigger and quick software trigger of frame grabbers.
LineScanSoftwareTrigger.cs Software Trigger by Line Scan Camera It shows how to get images via software trigger command by a line scan camera.

Sample Programs of Image Processing

File Name Sample Program Description
Image_Contrast.cs Adjust Image Contrast It shows how to adjust the image contrast.
Image_ConvertPixelType.cs Convert Image Format It shows how to convert the image format.
Image_Save.cs Save Image It shows how to save image data.
Image_HBDecode.cs Lossless Decoding It shows how to decode losslessly.
Image_Recording.cs Record It shows how to record videos.

Sample Programs of Parameter Configuration

File Name Sample Program Description
