Device Parameters Configuration

The SDK can access and configure the device parameters via GenICam Standard Protocol, and provides a set of APIs (IParameters) commonly used for cameras and frame grabbers to get and set parameter nodes of 6 types (integer, enumeration, float, boolean, string, and command).

Search for Node Name and Type

APIs of getting/setting nodes require configuring the node name and node type. Therefore, before calling these APIs, you need to get the node name and node type via search on the client.

  1. Connect to the device via the client.

  2. Select a certain feature parameter on the client feature tree to view the parameter node information below the feature tree.

Get/Set Node

  • Call IParameters.GetIntValue() to get integer nodes.
    Call IParameters.SetIntValue() to set integer nodes.
    string key = "Width";
    IIntValue intValue;
    // Get integer nodes
    device.Parameters.GetIntValue(key, out intValue);
    // Set integer nodes
    device.Parameters.SetIntValue(key, intValue.CurValue);
  • Call IParameters.GetEnumValue() to get enumeration nodes.
    Call IParameters.SetEnumValue() to set enumeration nodes.
    string key = "PixelFormat";
    IEnumValue enumValue;
    // Get enumeration node value
    device.Parameters.GetEnumValue(key, out enumValue);
    // Set enumeration node value
    device.Parameters.SetEnumValue(key, enumValue.CurEnumEntry.Value);
    // Set enumeration node value via string
    device.Parameters.SetEnumValueByString(key, enumValue.CurEnumEntry.Symbolic);
  • Call IParameters.GetBoolValue() to get boolean nodes.
    Call IParameters.SetBoolValue() to set boolean nodes.
    string key = "GammaEnable";
    bool boolValue;
    device.Parameters.GetBoolValue(key, out boolValue);
    // Set boolean node value
    device.Parameters.SetBoolValue(key, boolValue);
  • Call IParameters.GetFloatValue() to get float nodes.
    Call IParameters.SetFloatValue() to set float nodes.
    string key = "ExposureTime";
    IFloatValue floatValue;
    device.Parameters.GetFloatValue(key, out floatValue);
    // Set float node value
    device.Parameters.SetFloatValue(key, floatValue.CurValue);
  • Call IParameters.GetStringValue() to get string nodes.
    Call IParameters.SetStringValue() to set string nodes.
    string key = "DeviceUserID";
    IStringValue stringValue;
    device.Parameters.GetStringValue(key, out stringValue);
    // Set string node value
    device.Parameters.SetStringValue(key, stringValue.CurValue);
  • Call IParameters.SetCommandValue() to set command nodes.
    string key = "UserSetLoad";
    device.Parameters.SetCommandValue(key);
