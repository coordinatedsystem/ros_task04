Upgrades device via local file.

Parameters
handle[IN] It refers to the device handle.
strFilePathName[IN] It refers to the file name.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Remarks
  • Upgrade firmware can be sent to device by this API.
  • It may take a long response time since the API will only be recalled after the upgrade firmware is sent to device.
