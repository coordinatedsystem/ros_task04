Detailed Description

Includes APIs for reading files from and writing files to devices.

◆ MV_CC_FileAccessReadEx()

MV_CAMCTRL_API int __stdcall MV_CC_FileAccessReadEx ( IN void *  handle,
IN OUT MV_CC_FILE_ACCESS_EX *  pstFileAccessEx 
)

Reads the Data file from the device (extended).

Parameters
handle[IN] It refers to the device handle.
pstFileAccessEx[IN][OUT] It refers to the file access structure.
Returns
Returns MV_OK for success, and returns corresponding Error Code for failure.
Note
Call this API to read and write the file by using the buffer data in case of the error of no access permission. This API is the extended API of MV_CC_FileAccessRead().
