引入SDK的.Net程序集

开始前，请参照本文配置开发环境，然后将SDK的.Net程序集引入至您的项目并引用该库的命名空间。
SDK提供了基于.Net Standard Library 2.0框架、适配不同Windows系统的.Net程序集（MvCameraControl.NET.dll）。
在安装客户端后，您可从客户端安装路径下获取程序集，具体路径如下：..\Samples\64\C#\DotNet\AnyCPU\netstandard2.0。

开发环境要求

请确保开发环境已安装.Net框架，推荐使用.Net 5及以上框架。

操作步骤

  1. 将SDK的.Net程序集（MvCameraControl.Net.dll）引入至您的项目。

    以使用VSCode开发为例，直接编辑项目文件，添加程序集引用即可。

    <ItemGroup>
    <Reference Include="MvCameraControl.Net">
    <HintPath>..\..\..\Dotnet\AnyCPU\netstandard2.0\MvCameraControl.Net.dll</HintPath>
    </Reference>
    </ItemGroup>
  2. 引用MvCameraControl.Net.dll的命名空间 MvCameraControl 。
    using MvCameraControl;
    注意
    MvCameraControl.Net.dll中包含两个命名空间， MvCameraControl 和 MvCamCtrl.NET 。其中的 MvCamCtrl.NET 为兼容旧版本接口而保留，开发时无需关注该命名空间。本文档仅针对 MvCameraControl 说明引用该命名空间后的开发方法。
