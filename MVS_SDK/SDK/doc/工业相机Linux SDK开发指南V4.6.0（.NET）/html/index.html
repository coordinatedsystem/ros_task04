概述

工业相机SDK（.NET）是专门配合工业相机、采集卡、光源控制器、光源控制卡等设备使用的软件开发包。您可基于该SDK轻松搭建满足特定场景需求的视觉应用程序，实现设备控制、设备参数配置、获取数据、保存数据等功能。
该SDK兼容GigE Vision、USB3 Vision、Camera Link、CoaXPress和XoFlink协议。

运行环境

Linux 64位：Ubuntu 16.04 - 24.04 (64)、Redhat7 (64)、Centos7 (64)。
注解
若使用采集卡，只支持Ubuntu 16.04 - 24.04 64bit。
.NET环境要求：.NET 5及以上

使用说明

工业相机客户端软件默认安装在 /opt/（软件名称）路径下。您也可指定路径进行安装。
安装路径下包含有Development文件夹，该文件夹包含SDK的依赖文件。该文件夹目录结构如下：
  • bin（编译后的成果物）
  • doc（二次开发文档资料，包括SDK开发指南和Demo使用说明等）
  • includes（头文件）
  • lib（动态链接库）
  • Samples（各种编程语言的示例代码以及.NET程序集）
注意
不同版本的客户端开发目录文件可能存在差异，请以实际情况为准。
工业相机Linux版本SDK安装包包含了driver（驱动）、logserver（相机日志服务），也可以从该路径下获取。
