◆ LocalUpgrade()

Int32 IInterface.LocalUpgrade ( String  filePath)

设备本地升级

参数
filePath升级文件路径
返回
成功，返回MV_OK；失败，返回错误码

通过该接口可以将升级固件文件发送给设备进行升级。该接口需要等待升级固件文件成功传给设备端之后再返回，响应时间可能较长。
