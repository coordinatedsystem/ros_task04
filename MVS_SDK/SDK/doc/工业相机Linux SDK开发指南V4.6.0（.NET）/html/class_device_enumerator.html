详细描述

设备枚举类，支持枚举GigE Vision、USB3 Vision相机，及采集卡上的相机（GigE Vision、CameraLink、CoaXPress、XoFlink）

◆ EnumDevices()

static Int32 DeviceEnumerator.EnumDevices ( DeviceTLayerType  TLayerType,
out List< IDeviceInfo >  devInfoList 
)
static

枚举设备

参数
TLayerType设备接口类型
devInfoList设备列表
返回
成功，返回MV_OK；失败，返回错误码
注解
  • 设备接口类型为 MvGigEDevice 时，枚举所有GigE设备，包含虚拟GigE设备和采集卡上的GigE设备。
  • 设备接口类型为 MvUsbDevice 时，枚举所有USB设备，包含虚拟USB设备。
注意
设备列表的内存是在SDK内部分配的，调用该接口时会进行设备列表内存的释放和申请，请避免多线程枚举操作。
后续调用
枚举到设备后，通过DeviceFactory创建设备实例。
示例
Events_Camera.cs, GigE_ForceIP.cs, Grab_ActionCommand.cs, Grab_Callback.cs, Grab_ChunkData.cs, Grab_GetImageBuffer.cs, Grab_GigEMultiCast.cs, Grab_ImageClone.cs, Grab_Strategy.cs, Image_Contrast.cs, Image_ConvertPixelType.cs, Image_HBDecode.cs, Image_Recording.cs, Image_Save.cs, LineScanSoftwareTrigger.cs, MultiLightCtrl.cs, MultiLightCtrl_ImageStitching.cs, ParameterCamera_FileAccess.cs, ParameterCamera_LineScanIOSettings.cs, ParameterCamera_LoadAndSave.cs, ParametrizeCamera_AreaScanIOSettings.cs , 以及 Reconnect.cs.
