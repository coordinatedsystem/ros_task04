配置开发环境

在您开始二次开发前，请参照本文配置开发环境，将SDK引入至您的项目。

开发环境要求

请确保开发环境符合如下要求：
  • gcc/g++版本在4.8.5及以上，且已安装Make命令。
  • 已安装客户端。

操作步骤

  1. 在您的项目中，引入SDK的头文件。

    安装客户端后，您可在 /opt/（安装目录）/include 文件夹下，找到头文件。

    您可使用 –I 命令，指定头文件的路径。

    • 通过绝对路径引用头文件：
      -I/opt/（安装目录）/include/
    • 通过当前工程和include目录的相对路径引用头文件：
      -I../../../include/

  2. 在您的项目中，引入SDK的库文件。

    安装客户端后，您可在 /opt/（安装目录）/lib 文件夹下，找到库文件。

    注解
    您可使用env命令，查看环境变量，如下图所示。

    您可通过“-Wl,-rpath”命令，指定库文件的目录，并通过“-L”命令，连接库文件。

    • 通过绝对路径引用库文件：
      -Wl,-rpath=/opt/（安装目录）/lib/64 -L/opt/（安装目录）/lib/64 –lMvCameraControl
    • 通过环境变量引用库文件：
      -Wl,-rpath=$(MVCAM_COMMON_RUNENV)/64 -L$(MVCAM_COMMON_RUNENV)/64 –lMvCameraControl

  3. 使用make命令，编译程序。
