示例程序概览

安装客户端后，您可从安装路径下获取对应示例程序文件。您可参照提供的示例，在您的程序中实现对应功能。
注解
以64位为例，示例程序的具体路径为安装路径下的/Samples/64/C++/。
示例文件夹内包含如下类别的示例：
  • 基础示例
  • 采集卡示例
  • 面阵相机示例
  • 线阵相机示例

基础示例

以下为基础示例，您可从Samples文件夹下的General文件夹中获取对应的示例。
相机连接基础示例
文件 示例 描述
ConnectSpecCamera.cpp 通过IP地址连接相机 演示通过相机IP地址和相机对应的网卡IP地址，实现将相机连接至您的程序。
注解
输入的IP地址格式为xx.xx.xx.xx。
ForceIPEx.cpp 强制设置相机IP 演示强制设置网口相机IP，保证断电后重新连接，重新枚举后依旧生效。
ReconnectDemo.cpp 相机重连 演示相机掉线后，在心跳时间内重新连接相机。


图像获取基础示例
文件 示例 描述
Grab_Asynchronous.cpp 异步取图及处理 演示进行异步取图和图像处理。
注解
异步操作可规避同步时可能存在的耗时长问题。
GrabImage.cpp 获取相机图像 演示通过主动取图的方式获取相机图像。
GrabImage_Callback.cpp 通过回调获取相机图像 演示通过回调的方式获取相机图像。
GrabImage_HighPerformance.cpp 高性能获取相机图像 演示高效率的主动取图方式。
GrabMultipleCamera.cpp 多相机取流 演示实现多相机取图。您可以创建多个相机句柄handle以区分不同相机。


图像处理基础示例
文件 示例 描述
ChunkData.cpp 获取Chunk信息 演示打开ChunkData功能、配置ChunkData以及获取ChunkData信息。ChunkData例如Timestamp、Exposure 等。
Display.cpp 显示相机图像 演示调用显示图像接口，并实时显示相机采集到的图像数据。
HighBandwidthDecode.cpp 高清无损压缩 演示通过高清无损压缩算法将从相机取到的无损压缩码流解码成裸数据。
ImageProcess.cpp 图像处理 演示转换图像像素格式和保存图像。
ImageSave.cpp 保存相机设备图像 演示保存从相机设备拿到的图像。可保存的图像格式包含裸图(.raw)、jpeg、bmp、tiff、png。


参数设置基础示例
文件 示例 描述
MultiCast.cpp 设置组播模式 演示设置传输模式为组播模式。通过询问用户启动多播控制应用程序或者多播监控应用程序，并以指定的访问权限打开相机和设置组播模式。
ParametrizeCamera_FileAccess.cpp 导入或导出文件到相机 演示导入文件到相机、导出相机文件、以及获取文件存取进度。
ParametrizeCamera_LoadAndSave.cpp 导入或导出相机属性 演示将相机属性树的设置导出到ini文件（例如FeatureFile.ini）和从ini导入到相机属性树中。


事件基础示例
文件 示例 描述
Events.cpp 获取相机事件 演示配置相机事件功能、注册事件回调以及在事件回调函数中处理获取到的事件信息。Event事件例如：ExposureEnd。


采集卡示例

以下为采集卡基础示例，您可从Samples文件夹下的FrameGrabber文件夹中获取对应的示例。
文件 示例 描述
Events_Interface.cpp 打开并接受采集卡事件 演示打开并接收采集卡事件。
InterfaceAndDeviceDemo.cpp 采集卡下的相机控制 演示通过普通枚举接口枚举到自研采集卡下的相机，并实现打开相机、开始取流等操作。
InterfaceDemo.cpp 采集卡的枚举及控制 演示通过枚举接口枚举自研采集卡，并实现打开采集卡、设置属性等操作。
QuickSoftwareTrigger.cpp 采集卡软触发 演示使用采集卡实现快速软触发功能。快速软触发仅被部分采集卡支持，请以采集卡实际情况为准。


面阵相机示例

以下为面阵相机示例，您可从Samples文件夹下的AreaScanCamera文件夹中获取对应的示例。
文件 示例 描述
ParametrizeCamera_AreaScanIOSettings.cpp 面阵相机触发节点IO 演示配置面阵相机触发相关节点IO。
SetParam.cpp 设置相机参数 演示使用通用接口设置相机参数，包括Int、Float、Enum、Bool等类型节点的设置。
Trigger_Image.cpp 触发模式获取相机图像 演示触发模式下通过软触发主动取图的方式获取图像。
过程包含：创建句柄，打开相机，设置触发模式，设置触发源（软触发），开始取流，创建取流线程（发送软触发command命令， MV_CC_GetOneFrameTimeout() 取图），停止取流，关闭相机，销毁句柄。
Trigger_ImageCallback.cpp 触发模式下通过回调获取相机图像 演示通过软触发回调取图的方式获取图像。创建句柄，打开相机，设置触发模式，设置触发源（软触发），注册抓图回调函数，开始取流，在回调函数中可以获取通过软触发得到的图像信息，停止取流，关闭相机，销毁句柄。


线阵相机示例

以下为线阵相机示例，您可从Samples文件夹下的LineScanCamera文件夹中获取对应的示例。
文件 示例 描述
LineScanSoftwareTrigger.cpp 线阵相机软触发取图 演示线阵相机如何通过软触发命令获取图像。
MultiLightCtrl_ImageStitching.cpp 线阵相机分时曝光并拆图拼图 演示设置线阵相机采用2组分时曝光参数，先拆图再自上而下拼图，最后将拼好的raw图以bmp格式保存至本地。
ParametrizeCamera_LineScanIOSettings.cpp 线阵相机IO设置 演示通过 MV_CC_SetEnumValue() 配置线阵相机帧触发和行触发相关节点IO。
